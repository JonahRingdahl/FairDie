# FairDie

A browser-based chi-squared goodness-of-fit test for dice fairness. No build step, no server, no dependencies beyond Chart.js (loaded from CDN).

## Usage

Open `index.html` in a browser in one of two modes:

- **Record** — click die-face buttons to tally rolls, results update live. Rolls auto-save to localStorage and restore on page reload.
- **Upload** — drag-and-drop or browse a CSV, JSON, or plain-text file of roll data.

Click **Save rolls** to name and persist a die's tally to localStorage. Saved dice appear in a list below the recorder — Load restores them for further rolling, Delete removes them. Saving with an existing name+sides overwrites that entry.

Results show the chi-squared statistic, degrees of freedom, p-value, a fair/biased verdict, a bar chart of observed vs expected frequencies, and a face ranking.
